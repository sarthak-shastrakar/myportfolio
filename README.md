# 🚀 Sarthak Shastrakar - Personal Portfolio

Welcome to the source code of my personal portfolio website! This project showcases my skills, experience, and projects as a Full-Stack MERN Developer and Computer Engineering student. 

The website is built with a focus on modern aesthetics, smooth micro-interactions, and a highly responsive design.

## ✨ Features
- **Dynamic 3D UI Elements**: Interactive and beautiful space-themed mascot and graphics.
- **Smooth Animations**: Powered by `framer-motion` for page transitions, hover effects, and scroll interactions.
- **Modern Design**: Built with Tailwind CSS utilizing glassmorphism, gradient text, and a sleek dark mode appearance.
- **Responsive Navigation**: A custom magnetic tilt nav item for desktop and a slide-out drawer for mobile.
- **Component-based Architecture**: Modular React components for easy maintenance and scaling.

## 🛠️ Technologies Used
- **Frontend Framework**: React 19 (via Vite)
- **Routing**: React Router DOM v7
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📦 Local Development Setup

To run this project locally, follow these steps:

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-repo-url>
   cd Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Open `http://localhost:5173` to view it in the browser.

## 📜 Available Scripts

In the project directory, you can run:
- `npm run dev` - Runs the app in development mode.
- `npm run build` - Builds the app for production to the `dist` folder.
- `npm run preview` - Locally preview the production build.
- `npm run lint` - Runs the linter (Oxlint) to check for code quality.
